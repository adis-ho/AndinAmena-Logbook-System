# Amena Logbook System

Management sistem logbook transportasi untuk PT. Amena Safeeya Sejahtera.

## Tech Stack
- **Frontend Framework**: React 19 (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: React Context + TanStack Query
- **Utilities**: `xlsx`, `jspdf`, `date-fns`, `zod`

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build**
    ```bash
    npm run build
    ```
